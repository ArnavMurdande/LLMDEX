Methodology

LLMDEX is designed as a financial-grade analytics index. Every ranking decision is documented, every data point is traceable, and no value is ever fabricated. Below is how the system works.

Data Sources

We aggregate data from 5 independent benchmark sources: LiveBench, LMSYS Chatbot Arena, Artificial Analysis, LLM-Stats, and Vellum. Each source is independently validated before inclusion. Source confidence weights range from 0.70 (LLM-Stats) to 0.95 (LMSYS Arena).

Performance Leaderboard

Pure intelligence ranking. Bias-corrected using confidence weighting: adjusted_performance = performance × (0.85 + 0.15 × coverage). Models with more benchmark coverage are not unfairly advantaged. No cost or speed influence.

Value Leaderboard

Composite: 50% Performance + 30% Cost Efficiency + 20% Speed. Missing components redistribute weight proportionally. Uses adjusted performance for fairness.

Efficiency Leaderboard

Efficiency = Performance / Blended Cost, normalized as percentile ranks to prevent chart distortion. Only models with adjusted performance ≥ 60 qualify. Free models receive maximum efficiency.

Bias Correction

Models with sparse data get a confidence penalty: confidence_factor = benchmarks_available / 5. This prevents models with a single high score from dominating the leaderboard unfairly.

Limitations

No single metric captures "intelligence." Benchmarks test narrow capabilities. Costs change frequently. Community sentiment is experimental and never influences ranking. Use LLMDEX as one input to your evaluation, not the only one.
